# Portafolio de Proyectos - Especialización en Ciencia de Datos e Inteligencia Artificial

## Propósito del Repositorio

Este repositorio contiene una selección de proyectos desarrollados durante la Especialización en Ciencia de Datos e Inteligencia Artificial (ECDIA) de la Universidad Tecnológica (UTEC). Cada proyecto refleja las habilidades y conocimientos adquiridos durante el programa, aplicando técnicas de análisis de datos, aprendizaje automático y deep learning para resolver problemas del mundo real.

Los proyectos demuestran competencia en diversas áreas, incluyendo análisis exploratorio de datos, preprocessing, visualización, modelado predictivo y deployment de soluciones basadas en machine learning.

---

## Proyectos

### Proyecto 1: Análisis Histórico de los Juegos Olímpicos

**Descripción:** Análisis exhaustivo de 120 años de historia de los Juegos Olímpicos (1896-2016), examinando patrones de participación de países, rendimiento atlético, tendencias de género y factores que influyen en la obtención de medallas.

**Técnicas:** Análisis exploratorio de datos, visualización estadística, análisis de correlaciones, tests de normalidad, análisis de componentes múltiples (ACM).

[Ver notebook](./120/FINAL_Presentación_Grupo02_Entrega_III.ipynb)

---

### Proyecto 2: Detección de Cascos de Seguridad con CNN

**Descripción:** Sistema de deep learning para clasificación binaria que detecta si trabajadores en obras de construcción utilizan cascos de seguridad. El modelo se basa en una Red Neuronal Convolucional (CNN) entrenada con el dataset SHEL5K.

**Técnicas:** Redes Neuronales Convolucionales (CNN), clasificación binaria, data augmentation, transfer learning.

[Ver notebook](./AI_Safety_Helmet_Detection/AI_Safety_Helmet_Detection.ipynb)

---

### Proyecto 3: Predicción de Clientes Potenciales

**Descripción:** Modelo predictivo para identificar qué leads tienen mayor probabilidad de convertirse en clientes pagos para ExtraaLearn, una startup EdTech. El análisis incluye identificación de factores clave que impulsan la conversión.

**Técnicas:** Árboles de decisión, Random Forest, GridSearchCV, análisis de importancia de features, ROC-AUC.

[Ver notebook](./Potential_Customer_Prediction/Learner%20Notebook%20-%20Full%20Code%20Version%20-%20Potential%20Customers%20Prediction.ipynb)
